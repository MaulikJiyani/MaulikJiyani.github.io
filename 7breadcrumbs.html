You Are Here!(Breadcrumbs)


A "Breadcrumb" is one type of navigator that shows the actual location of the user in website or web applications. Breadcrumb shows you where you are. Breadcrumb basically displays you a route from the main webpage to where you are. User can go to previous webpage by using breadcrumb.



Here picture shows the example of the breadcrumb. That website is flipkart. It's contain the feature of the breadcrumb. In flipkart, we can show we search about one product and in last in upper of the product name we can see the specific path of the product place.
