Focus On Thing


Focus on thing one of the important guideline of the user interface design. There are lots of example for this in real life. For example, if someone go to mall for buying some important things. So user will only first focus on the thing which he/she try to find and need to them. This is the example of the real life phenomenon.


Same as computer, in one website there are lots of content included but user will see only the information which they need.

Like in above picture, Student want to know their grade, then student only focus on the grade column in the menu rather then all column on the dropdown list.
