Proper Way to Search Tool


“Proper way to search tool” is a concept that shows the position of the search tool in the website. There is one guideline provide in the don’t make me think book, that is ‘A way to search’. Searching tool is very important in every website. By using search tool, user can find things quickly. It’s a time saving and useful.


Here we can show one example of search tool of best buy website. Even small website also contain the search tool. It's useful component for finding things quickly. For example, user wants to buy mobile phone then user have to just search mobile in search tool and user will get the all result of mobile phones.
